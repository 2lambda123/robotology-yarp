YARP2

Welcome to YARP2, the newer, minty-fresh version of YARP: Yet Another Robot Platform.
  • Online YARP documentation for users, containing the most important classes and topics.

  • Printable user manual (pdf) -- not beautiful, but in case you want something you can print out.

  • All documentation containing all documented classes and topics, without any filtering. Good for developers, but perhaps a bit overwhelming.

  • Protocol specification (pdf) for YARP network activity.
