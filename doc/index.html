Welcome to YARP2, the newer, minty-fresh version of YARP: Yet Another Robot Platform.

Documentation

  • Online YARP documentation for users, containing the most important classes and topics.

  • ZIP file of YARP documentation for users, for reading locally.

  • User manual (pdf) -- a subset of our online documentation, packed into a single document. If you plan on printing this out we suggest you use a version with pagenumbers (pdf) rather than clickable links.

  • All documentation -- all documented classes and topics, without any filtering. Good for developers, but perhaps a bit overwhelming.

  • Protocol specification (pdf) for YARP network activity.

Installation

YARP source code is available (you need CMAKE to build these packages):
  • yarp-2.0.7.tar.gz for UNIX
  • yarp-2.0.7.zip for Windows
You can install YARP2 from CVS. You might want to check if it is passing its regression tests:
  • Most recent Windows report
  • Most recent Linux report
We have an unofficial Debian source package (all architectures and binary package (just i386). Add the following to your /etc/apt/sources.conf: 
   deb http://yarp0.sourceforge.net/downloads/yarp/debian stable main
(unstable is also available). Then you can set up YARP with: 
   sudo apt-get update
   sudo apt-get install yarp
If you want the source code and not just executables and header files, add the following: 
   deb-src http://yarp0.sourceforge.net/downloads/yarp/debian stable main
Then you can get the YARP source code with: 
   sudo apt-get update
   apt-get source yarp
