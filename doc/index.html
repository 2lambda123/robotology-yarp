Installation

You can install YARP from CVS: 
  cvs -d:pserver:anonymous@yarp0.cvs.sourceforge.net:/cvsroot/yarp0 login
  cvs -z3 -d:pserver:anonymous@yarp0.cvs.sourceforge.net:/cvsroot/yarp0 co -P yarp2
(use empty/blank password).

Or you can download a recent source code package:

  • yarp-2.1.2.tar.gz for UNIX
  • yarp-2.1.2.zip for Windows
You need CMake to build these packages.

You can check if YARP is passing its regression tests on your OS:

  • Most recent Windows / Linux / Darwin report.

Debian package

We have an unofficial Debian source package (all architectures) and binary package (just i386). Add the following to your /etc/apt/sources.conf: 
   deb http://eris.liralab.it/yarp/downloads/yarp/debian stable main
(unstable is also available). Then you can set up YARP with: 
   sudo apt-get update
   sudo apt-get install yarp
If you want the source code and not just executables and header files, add the following: 
   deb-src http://eris.liralab.it/yarp/downloads/yarp/debian stable main
Then you can get the YARP source code with: 
   sudo apt-get update
   apt-get source yarp
